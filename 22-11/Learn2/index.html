Bootstrap là framework cho frontend
        Nó bao gồm html , css , javascript cho phép dựng 1 giao diện web
        nhanh chóng và hỗ trợ responsive
        1 - Cài đặt bootstrap
        có 2 cách
        Cách 1 : sử dụng bootstrap CDN
        tạo ra 1 thẻ link có href là : https://stackpath.bootstrapcdn.com/bootstrap/4.3.1/css/bootstrap.min.css
        Cách 2 : download bootstrap về máy
        Download file bootstrap.min.css từ đường link : https://stackpath.bootstrapcdn.com/bootstrap/4.3.1/css/bootstrap.min.css
        về máy sau đó nhúng vào file html bằng thẻ link
        Phiên bản hiện tại của bootstrap là bản 4
        Link : https://getbootstrap.com/
        Tài liệu hướng dẫn : https://getbootstrap.com/docs/4.3/getting-started/introduction/
        Ví dụ về BS : https://getbootstrap.com/docs/4.3/examples/
